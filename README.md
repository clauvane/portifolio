# portifolio

## Overview
Sistema para gerenciar o portifolio de uma empresa.

## How to run

### Step 1
Execute no terminal
```
docker-compose up -d
```

### Step 2
Execute no terminal
```
./gradlew clean build
```

### Step 3
Execute no terminal
```
./gradlew bootrun
```

### Step 4
Acesse o browser
```
http://localhost:8080
```
